### 题型总结

CCF-CSP 认证考试的第一题主要以语法题与小模拟题为主，难度与C语言系列课程差不多甚至略低，在这38道真题中，基本没有涉及到任何算法和数据结构，要完成第一题，只需要将C语言课稍微复习一下即可。

涉及到了少数大家可能陌生的知识点：排序、多关键字比较/排序、简单贪心、简单思维题、字符串比较。我们在讲这些题目的时候，会讲这些知识点详细地给大家讲一下。

```cpp
char S[N],c;
int main() {
	scanf("%s",S);//读入字符串，读到空格或者回车结束 
	fgets(S,sizeof(S),stdin);//读入一行字符串，读到最后的回车结束 （会读回车） 
	scanf("%c",&c);//读入一个字符 
	
	//常用函数 <string.h>
	//strlen
	int n=strlen(S);//求字符串长度，不会算终止符号'\0' S[0~n-1] 
	
	//如果你习惯下标从1开始，可以写成
	scanf("%s",S+1);
	n=strlen(S+1);
	
	//char * strcpy ( char * destination, const char * source );
	//把source复制到 destination中，会复制 "\0"
	strcpy(S1,S2);//string copy
	strcpy(S1+1,S2+1);//都从1开始 
	strcpy(S1,"AABBCC"); 
	S1="AABBCC";
    
	//int strcmp(const char *str1, const char *str2)
	//比较str1与str2两个字符串的大小
	//str1==str2 返回值0;  str1>str2 返回值1; str1<str2 返回 -1
	strcmp(S1,S2); //1
	strcmp(S1,"AABB");
	strcmp("aabb","AABB");  //-1
}
```
